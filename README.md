# python-file-automation-toolkit

A collection of Python utilities designed to automate repetitive business tasks involving Excel files, CSV reports and operational data.

## Purpose

This project was created to demonstrate practical automation techniques that improve productivity in business environments.

The toolkit focuses on tasks such as:

- Merging CSV files.
- Cleaning Excel spreadsheets.
- Comparing reports.
- Detecting duplicate records.
- Validating data.
- Generating summary reports.

## Technologies

- Python
- Pandas
- OpenPyXL
- Regular Expressions
- CSV
